## 6. Бизнес-логика и алгоритмы расчётов

### 6.1 Основные принципы системы
1. Виртуальный счёт всегда соответствует реальному банковскому счёту
2. Резервирования не создают «денег из воздуха» — деньги либо на счёте, либо зарезервированы
3. Свободный остаток показывает реально доступные для трат средства
4. Месяц не может быть закрыт с отрицательным Свободным остатком

### 6.2 Формулы расчёта

```
Доход (план) = Σ(Основной доход всех участников за период) 
             + Σ(Разовые доходы текущего месяца)

ОЕП (план)   = Σ(Все активные статьи ОЕП в текущем месяце)

ПП (план)    = Σ(Ежемесячные взносы по активным статьям ПП)

Накопления (план) = Σ(Ежемесячные взносы по активным накоплениям)

Виртуальный счёт =
    Начальный баланс (перенесённый из предыдущего месяца)
  + Σ(Все подтверждённые доходы месяца)
  − Σ(Все подтверждённые ОЕП)
  − Σ(Списанные цели ПП и Накоплений)
  − Σ(Списанный свободный остаток)

Зарезервированные средства =
    Σ(Все подтверждённые, но ещё не списанные взносы по ПП и Накоплениям)

Свободный остаток = Виртуальный счёт − Зарезервированные средства
```
